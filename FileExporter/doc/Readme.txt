----
コマンド

	エクスポート

		FileExporter.exe /E R-DIR E-DIR SUCCESSFUL-FILE

			R-DIR ... 存在するディレクトリ、配下のファイルを E-DIR に移動する。
			E-DIR ... 存在する空のディレクトリ
			SUCCESSFIL-FILE ... 成功時ファイルを作成する。

			★ R-DIR 内のファイルを「移動」することに注意！

	インポート

		FileExporter.exe /I E-DIR SUCCESSFUL-FILE

			E-DIR ... 存在するディレクトリ
			SUCCESSFIL-FILE ... 成功時ファイルを作成する。
