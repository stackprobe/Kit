----
コマンド

	DirTime.exe [/C CREATE-TIME] [/W WRITE-TIME] [/A ACCESS-TIME] [/-] PATH [CREATE-TIME [WRITE-TIME [ACCESS-TIME]]]

		CREATE-TIME, WRITE-TIME, ACCESS-TIME ...

			YYYYMMDDhhmmssLLL 形式の文字列
			例: 20160215094730123
			全て省略した場合、現在セットされている日時を表示する。

		/- より前は順不同

		PATH ... ディレクトリ又はファイル

----
実行例

	DirTime.exe /C 19991231235959999 /W 20000101000000000 /A 20160215094730123 C:\Dir

	DirTime.exe C:\Dir\File.dat 19991231235959999 20000101000000000 20160215094730123
