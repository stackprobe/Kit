----
コマンド

	ImgTools.exe (/F 入力ファイル | /RWF 入出力ファイル | [/RF 入力ファイル] /WF 出力ファイル) 加工オプション...

		/F   ... 入力ファイルを指定して、入力ファイル名の後ろに適当なサフィックスを付けて出力する。
		/RWF ... 入力ファイルを指定して、入力ファイルを上書きする。
		/RF  ... 入力ファイルを指定する。省略すると、適当な画像からスタートする。
		/WF  ... 出力ファイルを指定する。

		入力ファイルのフォーマット = 何でも
		出力ファイルのフォーマット = png

----
加工オプション

	/E 幅 高さ

		現在の画像を指定された幅・高さに拡大・縮小する。

			幅   ... int 1〜
			高さ ... int 1〜

	/EW 幅

		現在の画像を指定された幅に縦横比を維持して拡大・縮小する。

			幅 ... int 1〜

	/EH 高さ

		現在の画像を指定された高さに縦横比を維持して拡大・縮小する。

			高さ ... int 1〜

	/C 左 上 幅 高さ

		指定された範囲を切り取り、現在の画像にする。

			左   ... int 0〜
			上   ... int 0〜
			幅   ... int 1〜
			高さ ... int 1〜

	/TW

		右上と左下を反転

	/T

		上下反転

	/R 1

		時計回りに 90 度回転

	/R 2

		時計回りに 180 度回転

	/R 3

		時計回りに 270 度回転

	/TLR

		左右反転

	/M 2

		下方向に鏡っぽく拡張

	/M 4

		左方向に鏡っぽく拡張

	/M 6

		右方向に鏡っぽく拡張

	/M 8

		上方向に鏡っぽく拡張

	/TP X_POS Y_POS

		指定位置のドットと同じ色のドットを透明 (A = 0) にする。

			X_POS ... int 0〜
			Y_POS ... int 0〜

	/TC R G B

		指定色と同じ色のドットを透明 (A = 0) にする。

			R ... int 0〜255
			G ... int 0〜255
			B ... int 0〜255

	/ROT ANGLE

		角度を指定して画像を回転する。

			ANGLE ... double ラジアン角

	/ROTEX ANGLE X Y A R G B D

		角度と「回転軸の座標」と「領域外の色」と「精度」を指定して画像を回転する。

			ANGLE ... double ラジアン角
			X ... double 回転軸の座標, 画像の中心は幅(高さ)÷２
			Y ... double 〃
			A ... int 0〜255 領域外の色
			R ... int 0〜255 〃
			G ... int 0〜255 〃
			B ... int 0〜255 〃
			D ... int 1〜 大きい程高精度, デフォルト = 16

	/EXTEND L T R B

		上下左右各方向にデフォルト色で拡張する。

			L ... int 0〜 左方向のドット数
			T ... int 0〜 上方向のドット数
			R ... int 0〜 右方向のドット数
			B ... int 0〜 下方向のドット数

	/EXTENDEX L T R B A R G B

		上下左右各方向に指定色で拡張する。

			L ... int 0〜 左方向のドット数
			T ... int 0〜 上方向のドット数
			R ... int 0〜 右方向のドット数
			B ... int 0〜 下方向のドット数
			A ... int 0〜255 指定色
			R ... int 0〜255 〃
			G ... int 0〜255 〃
			B ... int 0〜255 〃

	/PLAINS

		現在の画像をデフォルト色の画像にする。

	/PLAIN W H

		現在の画像を破棄して、指定サイズ・デフォルト色の画像にする。

			W ... int 1〜 幅
			H ... int 1〜 高さ

	/PLAINEX W H A R G B

		現在の画像を破棄して、指定サイズ・指定色の画像にする。

			W ... int 1〜 幅
			H ... int 1〜 高さ
			A ... int 0〜255 指定色
			R ... int 0〜255 〃
			G ... int 0〜255 〃
			B ... int 0〜255 〃

	/BOKASHI L T R B LEVEL

		矩形領域をぼかす

			L ... int 0〜
			T ... int 0〜
			R ... int 0〜
			B ... int 0〜
			LEVEL ... int 0〜

	/2 入力ファイル

		サブ画像を読み込む。

	/2X

		現在の画像とサブ画像を入れ替える。

	/PASTE 左 上

		現在の画像にサブ画像を貼り付ける。

			左 ... int 0〜 貼り付け位置
			上 ... int 0〜 〃

	/PRTSC

		プライマリモニタのイメージを現在の画像にする。

	/DOTFLTR A R G B

		ドット単位で画像フィルタを掛ける。

		A, R, G, B にはそれぞれ以下のいずれかを指定する。

			0 〜 255    ... 即値指定
			A           ... A の値
			R           ... R の値
			G           ... G の値
			B           ... B の値
			AVG_RGB     ... R, G, B の平均値
			A_REV       ... 255 - A
			R_REV       ... 255 - R
			G_REV       ... 255 - G
			B_REV       ... 255 - B
			AVG_RGB_REV ... 255 - AVG_RGB
			A:nn        ... A の値の nn パーセント (n は 0 〜 9) (ex. A:01, A:50, A:99)
			R:nn        ... R の値の nn パーセント (n は 0 〜 9) (ex. R:01, R:50, R:99)
			G:nn        ... G の値の nn パーセント (n は 0 〜 9) (ex. G:01, G:50, G:99)
			B:nn        ... B の値の nn パーセント (n は 0 〜 9) (ex. B:01, B:50, B:99)
