----
コマンド

	用紙サイズ名リストを表示する。

		PrintImage.exe /SL

	プリンタ名リストを表示する。

		PrintImage.exe /PL

	印刷する。

		PrintImage.exe [/M 左余白 上余白 右余白 下余白] [/S 用紙サイズ名] [/P プリンタ名] (/F 画像ファイル名)...

			余白は整数を指定する。
			単位は 1/100 インチ

----
コマンド引数をファイルで指定する。

	PrintImage.exe //R 引数列ファイル

		引数列ファイル ... コマンド引数を行毎に記述する。文字コードは Shift_JIS
