Kit
===

C#で書いたツールの寄せ集めです。
各フォルダ直下の doc フォルダの下に Readme.txt があります。
実行形式は以下の場所からダウンロードできます。

	http://stackprobe.dip.jp/_kit


開発環境
--------

	Windows 7 Home Premium
	Visual C# 2010 Express

	このファイルの場所は C:\Dev\Kit\README.txt です。


フォルダの説明
--------------

Compress

	ファイル圧縮ツール

ImgToCsv

	各種画像ファイル・csv 変換ツール

ImgTools

	画像ファイル加工ツール

RSA

	RSA鍵生成・暗号化・復号ツール


補足
----

	コマンド引数は大文字・小文字を区別しない。ex. /A == /a
